Sensitivity Analysis of PROMETHEE II

Authors: Katerina Kabassi, Aristotelis Martinis

Published in: Applied Sciences, Vol. 11, No. 19, 2021

Abstract

The quality of content and the attractiveness of an environmental website can create an environmentally friendly attitude before one visits a secured area. This paper evaluates environmental websites using models like PROMETHEE II to understand their effectiveness and robustness in ranking and usability.

Introduction

The World Wide Web transitioned from static, non-interactive Web 1.0 into dynamic Web 2.0 environments. This evolution has influenced how websites are designed and evaluated for impact and functionality.

Future Trends

The paper explores how semantic web technologies can help better structure unstructured data and how models like PROMETHEE II can offer scalable approaches for web evaluation and improvement.

Conclusion

Web applications are continually evolving, focusing on improving user engagement and managing unstructured data effectively. This paper highlights the significance of models like PROMETHEE II in these advancements.
